# Simple CRUD App

I just made a simple CRUD app that takes in a title and some information, and places it in a list below. The items created are then editable and deletable.

This app serves as a baseline on how I develop CRUD applications, with the intention of improving over with better practices.

Currently, the app stores the info until the page is refreshed.
